# SIMULATED BLACKJACK GAME
- DEALER VS. PLAYER
- REGULAR HIT, STAY, BUST, AND PUSH Options
- Text Based

## Rules:
1. Dealer must STAY on all 17's
2. When player has 5 cards in hand and value less than or equal to 21, player wins.
